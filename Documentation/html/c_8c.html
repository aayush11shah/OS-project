◆ job()

void* job ( void *  ptr)

Function that executes the actual job to be done by the process.

This function is run by pthread_create to execute whatever instruction a process is supposed to do. Using a switch case to check which process is running, it then performs the instructions and stops. the main function montiors the process using shared memory.

Returns
NULL pointer for use in pthread_create()
Parameters
ptrthe argument provided by pthread_create to the function: in this case, whether the process has finished execution or not

The main function performing the given task based on the process type.
